


# Flight-Delay-Data-Pipeline-Analysis

Dataset Link:[Link Text](https://www.kaggle.com/datasets/sriharshaeedala/airline-delay/data)

=====

“This project analyzes flight delay patterns across US airports and airlines using historical flight data. It includes both overall trends and specific insights for individual airports and carriers.”

=====
# Technologies Used:

Python
Pandas
Matplotlib, Seaborn
Jupyter Notebook

# Analysis Performed

Overall Analysis: Total delays, cancellations, diversions across all airports and carriers.
Per-Airport Analysis: Average delay per flight, total delay, top causes of delay.
Per-Carrier Analysis: Average delay per flight, total delay, top causes.
Monthly/Seasonal Trends: Peak delay months.
Visualizations
